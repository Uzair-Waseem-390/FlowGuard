## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Configuration

Make sure your backend API is running on `http://localhost:8000` (or update the API_BASE_URL in `src/utils/api.js`).
